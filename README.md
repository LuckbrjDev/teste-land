# Omega Software Portfolio

Landing page da Omega Software com portfólio de sistemas, ferramentas operacionais,
automações e páginas profissionais.

## Projetos em destaque

- Snow On Ice 3.0
- System Hunter
- Chamado Express 2.0
- CRM de Clientes
- Landing Pages Profissionais
- Omega ERP
- Calculadora ICMS 2026
- Ciot Lab
- Oni

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

O projeto está configurado para deploy na Vercel com Next.js.
